    投稿须知 / SUBMISSION GUIDELINES

    欢迎向 SCIRecycle 提交您的观察与思考。我们不以学历、头衔或所属机构作为评判标准，我们只在乎您的思想是否足够锐利，是否切中了现实的沉疴。

    1. 征稿范围

    我们特别欢迎（但不限于）以下领域的跨学科研究与观察：

    • 隐性劳动经济学：关于情绪劳动、家务外包以及那些“理所应当”的牺牲。
    • 日常微观权力学：解构爹味说教、规训话语以及职场/家庭中的隐性权力结构。
    • 空间与性别社会学：从日常痛点出发，探讨系统性的偏见与资源分配不均。
    • 学术与话语霸权批判：对论资排辈、圈子文化及学术资源垄断的实证分析。

    2. 格式与规范

    我们拒绝被僵化的学术格式绑架。思想的深度不应由参考文献的排版格式决定。

    • 行文风格：接受学术论文、田野调查报告、深度纪实甚至逻辑严密的杂文。
    • 引文要求：不强制要求引用核心期刊文献。社交媒体截图、日常对话录音、个人真实经历，均可作为强有力的一手参考文献。
    • 语言限制：请在行文中尽量避免使用不必要的晦涩学术黑话。我们致力于让所有人都能看懂并参与讨论，而不是制造新的理解壁垒。

    3. 评审机制

    本刊实行反说教同行评审 (Anti-Mansplaining Peer Review)：

    • 您的稿件不会被交由任何既得利益者或沉迷于父权视角的“权威”进行居高临下的审判。
    • 我们的评审侧重于：文章是否揭示了被遮蔽的真实？逻辑是否自洽？是否为弱势或被忽视的群体提供了理论武器？

    4. 如何投稿

    请将您的文章（Markdown 格式或纯文本）通过 GitHub Pull Request 提交，或直接发送至编辑部公开邮箱。

    我们期待在这里，看到那些无处安放的灵魂和被系统过滤掉的声音。
